# nezha-theme-mdui

自行制作的另一个哪吒面板 MDUI 主题。

## Preview

![服务器列表截图](/screenshot/server.png "服务器列表截图")
![服务列表截图](/screenshot/service.png "服务列表截图")

## Usage

1. 下载仓库到本地
2. 复制本仓库的 `theme-custom` 文件夹，替换你面板所在服务器的 `/opt/nezha/dashboard/theme-custom/` 文件夹
3. 登陆面板后台，选择主题为 `Custom(local)`
4. 刷新前台，应该就能看到效果了
